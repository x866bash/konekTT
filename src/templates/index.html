TikTok Login Kit (Flask)

Contoh aplikasi login dengan TikTok (OAuth 2.0). Silakan klik tombol di bawah.

Login with TikTok

Endpoint

  • /login/tiktok mulai login
  • /auth/tiktok/callback callback
  • /me user aktif
  • /users daftar user
  • POST /admin/refresh/<open_id> refresh satu user
  • POST /admin/refresh_all refresh semua user

Tips

  • Pastikan Redirect URI sama persis dengan yang didaftarkan.
  • Scope default user.info.basic (untuk tampilan & avatar).
  • Token kadaluarsa? Gunakan endpoint refresh di atas.
